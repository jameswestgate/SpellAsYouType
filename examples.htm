Example 2

Spell checking of a textarea element with a UK English dictionary.

$('#txtTextarea').spellayt({url: "dictionary.enGB.txt"});
View example

Example 3

Spell checking with dictionary loaded callback notification and synchronous spell checking.

//Event callbacks
$.fn.spellayt.loadError = function(textStatus) { alert("An error occurred loading the dictionary."); };
$.fn.spellayt.loaded = function() { $('#para').text("Dictionary loaded.") };

//Check spelling link
$('a#check').click(function() {
	           
    //Run synchronous spell checking
    $('#txtTextarea').spellcheck();

    return false;
});

//Set up spell checking as you type	        
$('#txtTextarea').spellayt({ url: "dictionary.enGB.txt" });
